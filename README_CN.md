## 🎬 您可以做什么

### 📊 **Excel操作综合题**

在原始文件中完成Excel考试任务，保留公式、筛选和图表：

**传统方式**：手动计算、手动筛选、手动制图
**本技能方式**：自动编写公式（`=C11-D11-E11`）、数据筛选、创建图表并引用数据源

```
输入：Excel考试题文件（.xls/.xlsx）
输出：完成文件，单元格中可见公式
成本：$0（本地处理）
时间：2分钟 vs 30分钟手工操作
```

### 📈 **学生综测分析**

计算平均分数，避免陷入重复平均陷阱：

**传统方式**：跨多个文件手动搜索、手动计算、可能对已平均数据再次平均
**本技能方式**：自动文件扫描、智能数据提取、跳过已平均值的智能平均值计算

```
输入：多个学期Excel文件
输出：准确的平均分数及详细分解
示例：86.26 + 85.54 + 81.36（跳过已平均的85.9）= 84.39
准确度：100%（无计算错误）
```

### 📉 **数据可视化**

创建专业图表和可视化：

**传统方式**：手动图表设置、复制粘贴数据、手动格式化
**本技能方式**：自动生成引用原始数据的图表、趋势分析、异常检测

```
输入：原始Excel数据
输出：正确引用数据的图表、统计报告、趋势分析
功能：柱状图、趋势线、异常高亮
```

## 🔧 高级功能

### **智能平均值计算（避免陷阱）**

```python
# ❌ 错误：平均已平均的数据
成绩 = [85.9, 81.36, 83.88]  # 85.9已经平均过！
平均 = sum(成绩) / 3 = 83.71  # 错误！

# ✅ 正确：使用原始学期数据
成绩 = [86.26, 85.54, 81.36]  # 原始学期数据
平均 = sum(成绩) / 3 = 84.39  # 正确！
```

**本技能自动跳过已平均列**以确保计算准确性。

### **Excel公式保留**

```python
# 编写公式，不是值
ws.cell(row=11, column=6).value = "=C11-D11-E11"  # 公式
# vs
ws.cell(row=11, column=6).value = 8  # 值（错误方法）
```

**用户可以在Excel中看到公式，理解逻辑，修改计算**。

### **跨平台兼容**

```python
# Windows
pip_path = venv_path / 'Scripts' / 'pip.exe'
python_path = venv_path / 'Scripts' / 'python.exe'

# Linux/Mac
pip_path = venv_path / 'bin' / 'pip'
python_path = venv_path / 'bin' / 'python'
```

**在所有平台统一工作**。
